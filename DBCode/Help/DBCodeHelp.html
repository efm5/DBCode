The Menu

The menu has XXX items: “File”; “Edit”; “View”; “Options”; and “Help”. All XXX of these items have accelerator keys. All menu items have either accelerator keys (when they have sub–subitems) and/or keyboard shortcuts.

The File Menu (ALT+F)

New (CTRL+n)

“New” clears all text from the text box; if there is any text already in the text box, you will be offered the chance to save it.
